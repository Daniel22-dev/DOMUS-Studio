DOMUS Studio v6.1.0

Vestavěný Test Lab a GitHub modularizace

Verze 6.1 nemění datové schéma projektů. Zaměřuje se na diagnostiku, udržovatelnost zdrojového kódu a automatické testování každé změny na GitHubu.

Test Lab přímo v aplikaci

  • tichý self-test po startu,
  • výsledky PASS, WARN a FAIL,
  • kontrola runtime, UI, bezpečnosti, dat, DOMUS Verify, záloh, IndexedDB a PWA,
  • bezpečný dočasný databázový round-trip,
  • stažitelný diagnostický protokol.

Modularizace

  • 13 doménových částí hlavní aplikace ve src/app,
  • 5 samostatných modulů ve src/core,
  • reprodukovatelný build a kontrola zastaralých generovaných souborů,
  • dokumentovaná architektura a pravidla pro pull requesty.

GitHub Actions

Při pushi a pull requestu proběhne sestavení, jednotkové, bezpečnostní, statické, PowerShell, prohlížečové, HTTP a IndexedDB testy.

Kompatibilita

Datové schéma zůstává ve verzi 6. Starší projekty a zálohy zůstávají kompatibilní. Koncový uživatel nepotřebuje Node.js.
