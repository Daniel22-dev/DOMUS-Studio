DOMUS Studio Premium v7.0

Verze 7 dokončuje prémiové etapy A–C.

Etapa A

Onboarding, šablony, značka, designové režimy, Report Studio, rozšířený Test Lab, GitHub CI a Tauri desktop.

Etapa B

Precision 2D s magnetickými body, vrstvami, vícenásobným výběrem, Undo/Redo, parametrickou knihovnou, kótami a DXF.

Etapa C

RealSpace 3D přes Three.js, PBR materiály, stíny, řezy, fullscreen, PNG/GLB export, položkové slučování a výkonnostní worker.

Distribuce

GitHub Release workflow sestaví instalační balíčky a updater. Podpis aktualizací a Windows instalátoru vyžaduje vlastní klíče v GitHub Secrets.
